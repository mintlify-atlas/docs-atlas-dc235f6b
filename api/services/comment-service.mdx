The Comment Service enables users to create, read, update, and delete comments on anime episodes.

## Import

```javascript
import {
  getComments,
  addComment,
  updateComment,
  deleteComment
} from './services/commentService.js';
```

## API Endpoints

The Comment Service interacts with the following backend endpoints:

- `GET /api/comments/:animeId/:episodeId` - Get episode comments
- `POST /api/comments` - Add a new comment
- `PUT /api/comments/:id` - Update a comment
- `DELETE /api/comments/:id` - Delete a comment

## Functions

### getComments

Retrieves all comments for a specific anime episode.

```javascript
const result = await getComments('one-piece', 'episode-1120');

if (!result.error) {
  console.log(`Found ${result.data.length} comments`);
  result.data.forEach(comment => {
    console.log(`${comment.username}: ${comment.text}`);
  });
}
```

## Usage Example

Here's a complete example of a comment section implementation:

```javascript
import { getComments, addComment, updateComment, deleteComment } from './services/commentService.js';

class CommentSection {
  constructor(animeId, episodeId) {
    this.animeId = animeId;
    this.episodeId = episodeId;
    this.comments = [];
  }

  async load() {
    const result = await getComments(this.animeId, this.episodeId);
    if (!result.error) {
      this.comments = result.data;
      this.render();
    }
  }

  async post(text) {
    const result = await addComment({
      animeId: this.animeId,
      episodeId: this.episodeId,
      text: text
    });
    
    if (!result.error) {
      this.comments.unshift(result.data.comment);
      this.render();
    }
  }

  async edit(commentId, newText) {
    const result = await updateComment(commentId, { text: newText });
    if (!result.error) {
      const index = this.comments.findIndex(c => c.id === commentId);
      if (index !== -1) {
        this.comments[index] = result.data.comment;
        this.render();
      }
    }
  }

  async remove(commentId) {
    const result = await deleteComment(commentId);
    if (!result.error) {
      this.comments = this.comments.filter(c => c.id !== commentId);
      this.render();
    }
  }

  render() {
    const container = document.getElementById('comments');
    container.innerHTML = this.comments.map(comment => `
      <div class="comment" data-id="${comment.id}">
        <img src="${comment.avatar}" alt="${comment.username}">
        <div>
          <strong>${comment.username}</strong>
          <p>${comment.text}</p>
          <small>${new Date(comment.createdAt).toLocaleString()}</small>
        </div>
      </div>
    `).join('');
  }
}
```

## Error Handling

```javascript
const result = await addComment(commentData);

if (result.error) {
  switch (result.statusCode) {
    case 401:
      console.error('Please log in to comment');
      break;
    case 400:
      console.error('Invalid comment data');
      break;
    case 403:
      console.error('You can only edit/delete your own comments');
      break;
    case 429:
      console.error('Too many comments. Please wait before posting again.');
      break;
    default:
      console.error('Error:', result.message);
  }
}
```

<ResponseField name="error" type="boolean">
  True if an error occurred
</ResponseField>

<ResponseField name="message" type="string">
  Error message description
</ResponseField>

<ResponseField name="statusCode" type="number">
  HTTP status code
</ResponseField>

<Warning>
  Users can only update or delete their own comments. The backend enforces this with user ID verification.
</Warning>

<Note>
  All comment operations except `getComments` require the user to be authenticated.
</Note>
