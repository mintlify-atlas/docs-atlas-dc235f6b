Components in AnimeAho Client are **pure JavaScript functions** that create and return DOM elements. They follow a functional programming approach with clear inputs and outputs.

## Component Pattern

All components follow this basic structure:

```javascript
/**
 * Component documentation
 * @param {Object} data - Input data
 * @param {boolean} options - Configuration options
 * @returns {HTMLElement} DOM element
 */
export function componentName(data, options) {
    // 1. Create elements
    const element = document.createElement('div');
    
    // 2. Set attributes and styles
    element.className = 'component-class';
    
    // 3. Add content
    element.textContent = data.value;
    
    // 4. Attach event listeners if needed
    element.addEventListener('click', handleClick);
    
    // 5. Return the element
    return element;
}
```

## Core Components

### AnimeCard

The `AnimeCard` component renders anime as cards in a grid layout. It's one of the most reused components.

**Location:** `js/components/AnimeCard.js`

```javascript
export function renderAnimeCard(anime, badgeText = null, isDetailView = true) {
    const article = document.createElement('article');
    article.className = 'anime-card';
    article.setAttribute('data-anime-slug', anime.slug);

    const imageLink = document.createElement('a');
    imageLink.className = 'anime-card__a';
    imageLink.href = '#';

    const image = document.createElement('img');
    image.className = 'anime-card__image';
    image.src = anime.cover;
    image.alt = `Portada de ${anime.title}`;
    image.loading = 'lazy';

    imageLink.appendChild(image);

    // Optional badge (e.g., "Episode 3")
    if (badgeText) {
        const badge = document.createElement('div');
        badge.className = 'anime-card__badge';
        badge.textContent = badgeText;
        imageLink.appendChild(badge);
    }

    const titleElement = document.createElement('p');
    titleElement.className = 'anime-card__name';
    titleElement.textContent = anime.title;

    article.appendChild(imageLink);
    article.appendChild(titleElement);

    // Navigation handler
    imageLink.addEventListener('click', (event) => {
        event.preventDefault();
        handleCardNavigation(anime.slug, isDetailView);
    });

    return article;
}
```

**Key Features:**
- Lazy loading for images
- Optional badge overlay
- Configurable navigation (detail view vs episode view)
- Semantic HTML with `<article>` element

**Usage Example:**

```javascript
import { renderAnimeCard } from '../components/AnimeCard.js';

// Detail view card
const detailCard = renderAnimeCard(anime, 'TV', true);

// Episode card with badge
const episodeCard = renderAnimeCard(episode, 'Episodio: 12', false);
```

The `AnimeDetail` component renders the complete anime information page.

**Location:** `js/components/AnimeDetail.js`

```javascript
export function renderAnimeDetail(animeData) {

    renderEpisodesList(animeData.episode, animeData.cover);

```

**Architecture Features:**

1. **DOM Caching** - Caches references to avoid repeated queries:

```javascript
let domElementCache = {};

function cacheDOM() {
    domElementCache = {
        cover: document.getElementById('anime-cover'),
        title: document.getElementById('anime-title'),
        description: document.getElementById('anime-description'),
        genres: document.getElementById('anime-genres'),
        episodeList: document.getElementById('anime-list')
    };
}
```

2. **Modular Rendering** - Each section has its own render function:

```javascript
function renderRatingWithStars(rating) {
    const ratingNumber = parseFloat(rating) || 0;
    const container = document.createElement('div');

        starsContainer.appendChild(starSVG);

    domElementCache.popularity.appendChild(container);
}
```


### EpisodeList

Manages the rendering and interaction of episode lists.

**Location:** `js/components/EpisodeList.js`

```javascript
export function renderEpisodesList(episodeList, animeCoverUrl) {
    currentEpisodesList = episodeList;
    currentEpisodeSortOrder = 'desc';

    const sortedEpisodes = getSortedEpisodes();
    renderEpisodes(sortedEpisodes, animeCoverUrl);
    attachEpisodeSortListener(animeCoverUrl);
}
```

**Key Features:**

1. **Episode Card Creation:**

```javascript
export function createEpisodeCard(episode, animeCoverUrl) {
    const container = document.createElement('div');
    container.className = 'main__anime-episode';
    container.style.display = 'none'; // Hidden until image loads

    const link = createEpisodeLink(episode);
    const image = createEpisodeImage(animeCoverUrl, episode, container, link);
    const episodeNumber = createEpisodeNumber(episode.number);

    link.appendChild(image);
    container.appendChild(link);
    container.appendChild(episodeNumber);

    return container;
}
```

2. **Progressive Image Loading:**

```javascript
function createEpisodeImage(animeCoverUrl, episode, container, link) {
    const image = document.createElement('img');
    image.src = buildEpisodeCoverUrl(animeCoverUrl, episode.number);

    image.onload = () => {
        container.style.display = 'block';
    };

    image.onerror = async () => {
        const exists = await existEpisode(episode.number);
        if (!exists) {
            container.remove(); // Remove non-existent episodes
            return;
        }
        applyFallbackImage(image, link);
        container.style.display = 'block';
    };

    return image;
}
```

3. **Dynamic Sorting:**

```javascript
function getSortedEpisodes() {
    const sorted = [...currentEpisodesList];

    if (currentEpisodeSortOrder === 'asc') {
        sorted.sort((a, b) => a.number - b.number);
    } else {
        sorted.sort((a, b) => b.number - a.number);
    }

    return sorted;
}
```

## Component Communication

Components communicate through:

### 1. Function Parameters

```javascript
renderAnimeCard(anime, badgeText, isDetailView)
```

### 2. Event Listeners

```javascript
imageLink.addEventListener('click', (event) => {
    event.preventDefault();
    handleCardNavigation(anime.slug, isDetailView);
});
```

### 3. Module-Level State

```javascript
let listEpisodes = [];
let animeCover = '';

export function renderAnimeDetail(animeData) {
    listEpisodes = animeData.episode;
    animeCover = animeData.cover;
    // ...
}
```

## Component Lifecycle

```
┌─────────────────────────────────────┐
│   1. Component Function Called       │
│      with data parameters            │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   2. DOM Elements Created            │
│      (createElement, setAttribute)   │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   3. Content & Styles Applied        │
│      (textContent, className)        │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   4. Event Listeners Attached        │
│      (addEventListener)              │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   5. Element Returned                │
│      Ready for DOM insertion         │
└─────────────────────────────────────┘
```

## Best Practices

### 1. Single Responsibility

Each component does one thing well:

```javascript
// Good: Focused component
export function createEpisodeNumber(number) {
    const p = document.createElement('p');
    p.className = 'main__anime-episode-number';
    p.textContent = `Episodio ${number}`;
    return p;
}
```

### 2. Immutable Parameters

Don't modify input parameters:

```javascript
// Good: Create new sorted array
const sorted = [...currentEpisodesList];
sorted.sort((a, b) => a.number - b.number);
```

### 3. Error Handling

Handle edge cases gracefully:

```javascript
if (!episodeListContainer) return;

if (!response.data || response.data.length === 0) {
    section.innerHTML = '<p>No hay episodios disponibles.</p>';
    return;
}
```

### 4. Accessibility

Include proper ARIA labels and semantic HTML:

```javascript
image.alt = `Portada de ${anime.title}`;
link.setAttribute('title', `Ver episodio ${episode.number}`);
svg.setAttribute('aria-label', `Estrella ${position}`);
```

### 5. Performance Optimization

```javascript
// Use DocumentFragment for batch insertions
const fragment = document.createDocumentFragment();
animes.forEach(anime => {
    fragment.appendChild(renderAnimeCard(anime, anime.type, true));
});
container.appendChild(fragment);
```

## Testing Components

Components can be tested in isolation:

```javascript
// Example test structure
const mockAnime = {
    slug: 'test-anime',
    title: 'Test Anime',
    cover: 'https://example.com/cover.jpg'
};

const card = renderAnimeCard(mockAnime, 'TV', true);

// Verify structure
console.assert(card.tagName === 'ARTICLE');
console.assert(card.querySelector('img').src === mockAnime.cover);
```

<Note>
All components are designed to be **framework-agnostic** and rely only on standard DOM APIs, making them easy to test and maintain.
</Note>
