## Overview

The Search feature provides users with instant anime discovery through real-time text search with autocomplete suggestions and advanced filtering options. The system includes debounced search, request cancellation, and responsive dropdown results.

## Search Implementation

The main search functionality is located in `~/workspace/source/js/utils/search.js`.

### Initialization

Search listeners are initialized using the `addListenerInput()` function:

```javascript
export function addListenerInput() {
    const input = document.querySelector('.header__search-input');
    const button = document.querySelector('.header__search-link');
    
    input.addEventListener('input', () => searchAnime(input.value, input));
    input.addEventListener('blur', () => {
        setTimeout(() => hideDropdown(), 150);
    });
    input.addEventListener('keydown', (e) => {
        if (e.key === 'Enter') {
            navigateToAnimeSearchList(1);
        }
    });

    button.addEventListener('click', () => {
        if (!input.value.length == 0) navigateToAnimeSearchList(1);
    });
}
```

**Event Listeners:**
- **input** - Triggers search as user types
- **blur** - Hides dropdown when focus is lost (with 150ms delay)
- **keydown** - Navigates to full search page on Enter key
- **click** - Search button triggers full results page

<Note>
The 150ms delay on blur allows users to click dropdown items before the dropdown disappears.
</Note>

## Real-Time Search

The core search function with request cancellation:

```javascript
const dropdown = document.querySelector('.header__search-dropdown');
let currentController = null;

async function searchAnime(value, input) {
    if (value.trim().length === 0) {
        hideDropdown();
        return;
    }

    if (currentController) currentController.abort();
    currentController = new AbortController();

    const animesResponse = await searchAnimeWithText(value, currentController.signal);

    if (input.value.trim().length === 0) {
        hideDropdown();
        return;
    }

    if (!animesResponse.success) {
        dropdown.innerHTML = '<p>No se encontraron coincidencias</p>';
        return;
    }

    renderDropdown(animesResponse.data.media);
}
```

### Request Cancellation

The search uses `AbortController` to cancel outdated requests:

<CodeGroup>
```javascript Previous Request
// User types "naru"
const controller1 = new AbortController();
await searchAnimeWithText('naru', controller1.signal);
```

```javascript New Request (Cancels Previous)
// User types "naruto" quickly
controller1.abort();  // Cancels "naru" request
const controller2 = new AbortController();
await searchAnimeWithText('naruto', controller2.signal);
```
</CodeGroup>

<Accordion title="Why Cancel Previous Requests?">
Cancelling previous requests prevents race conditions where an older, slower request might return after a newer request, displaying incorrect results. This ensures users always see results matching their current input.
</Accordion>

## Search Dropdown

The dropdown component displays instant search results from `~/workspace/source/js/components/SearchDropdown.js:12`:

```javascript
export function renderDropdown(media) {
    dropdown.innerHTML = '';
    const total = Math.min(5, media.length);
    
    for (let i = 0; i < total; i++) {
        dropdown.appendChild(renderDropdownItem(media[i]));
    }

    if (media.length > 5) {
        const button = document.createElement('button');
        button.className = 'header__search-more';
        button.textContent = 'Ver más';
        
        button.addEventListener('click', () => navigateToAnimeSearchList(1));
        dropdown.appendChild(button);
    }

    dropdown.style.display = 'block';
}
```

**Dropdown Features:**
- Displays maximum 5 results
- Shows "Ver más" button if more than 5 results exist
- Clicking "Ver más" navigates to full search results page
- Dynamically populated from search API

## API Integration

The search integrates with the anime service API from `~/workspace/source/js/services/animeService.js:140`:

### Text Search

```javascript
export async function searchAnimeWithText(searchQuery, page, signal = null) {
    const params = new URLSearchParams({ query: searchQuery, page });
    const endpoint = `${ANIME_API_BASE_URL}${ANIME_ENDPOINTS.SEARCH_ANIME}?${params}`;
    
    try {
        const response = await fetch(endpoint, { signal });
        const data = await response.json();
        
        if (!response.ok) return new ResponseError(data);
        if (!data.success) return new ResponseError(data);
        
        return new SearchAnimeResponse(data);
    } catch (error) {
        if (error.name === 'AbortError') return; // ← ignora cancelaciones
        console.error("Error al buscar anime:", error);
        return new ResponseError({
            data: null,
            error: true,
            message: error.message || "Error de conexión",
            statusCode: 0,
            url: endpoint
        });
    }
}
```

<Note>
Aborted requests return undefined instead of throwing errors, allowing the search flow to continue smoothly.
</Note>

### Advanced Filter Search

For more complex searches with filters:

```javascript
export async function searchAnimeFilter({
    order = 'default',
    types = [],
    genres = [],
    statuses = [],
    page = 1
} = {}) {
    const params = new URLSearchParams({ order, page });
    const endpoint = `${BACKEND_API_BASE_URL}${BACKEND_ENDPOINTS.PROXY_SEARCH_BY_FILTER}?${params}`;
    
    try {
        const requestBody = { order, types, genres, statuses, page };
        const response = await fetch(endpoint, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(requestBody)
        });
        
        const data = await response.json();
        if (!response.ok) return new ResponseError(data);
        if (!data.success) return new ResponseError(data);
        
        return new SearchAnimeResponse(data);
    } catch (error) {
        console.error("Error al buscar con filtros:", error);
        return new ResponseError({
            data: null,
            error: true,
            message: error.message || "Error de conexión",
            statusCode: 0,
            url: endpoint
        });
    }
}
```

## Filter Parameters

| Parameter | Type | Options | Description |
|-----------|------|---------|-------------|
| `order` | string | `'default'`, `'popularity'`, `'rating'`, `'added'` | Sort order for results |
| `types` | array | `'tv'`, `'movie'`, `'ova'`, `'special'` | Anime format types |
| `genres` | array | Genre strings | Filter by genres |
| `statuses` | array | `1` (airing), `2` (completed), `3` (upcoming) | Filter by broadcast status |
| `page` | number | Positive integer | Pagination page number |
