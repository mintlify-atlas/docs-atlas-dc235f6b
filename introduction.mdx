## Architecture Overview

AnimeAho Client follows a modular architecture with clear separation of concerns:

- **Pages** - Individual HTML pages with associated JavaScript controllers
- **Components** - Reusable UI components for rendering content
- **Services** - API integration and data fetching logic
- **Utils** - Helper functions for common tasks
- **Models** - Data structures and type definitions

<Note>
AnimeAho is a client-side application that integrates with external anime APIs. It requires no backend deployment, making it easy to host as a static site.
</Note>

## Technology Stack

- **Vanilla JavaScript** - No framework dependencies, pure ES6+ modules
- **CSS3** - Custom styling with CSS variables for theming
- **HTML5** - Semantic markup for accessibility
- **External APIs** - AnimeFlv API for anime data
